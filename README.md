# 📊 پرتفولیوپلاس (PortfolioPlus) 
**سیستم مدیریت جامع سبد دارایی و سرمایه‌گذاری (بورس و فلزات گران‌بها)**

یک اسکریپت متن‌باز (Open Source) بر پایه معماری **PHP MVC** برای مدیریت یکپارچه و هوشمندانه سبد دارایی. این سیستم به شما اجازه می‌دهد سهام بورس ایران و بازار فلزات (طلا، نقره، سکه و مس) را در یک پنل واحد مدیریت کرده و سود و زیان خود را به صورت لحظه‌ای رصد کنید.

---

## ✨ امکانات کلیدی

- **📈 مدیریت چندگانه بازارها:** پشتیبانی همزمان از سهام بورس ایران، طلا، سکه، نقره و مس.
- **💰 محاسبه سود و زیان در لحظه:** محاسبه خودکار میانگین قیمت خرید، قیمت تمام شده، ارزش فعلی و سود/زیان (P&L) هر دارایی.
- **🏢 مدیریت چند کارگزاری:** امکان تفکیک دارایی‌ها بر اساس مبدا خرید یا کارگزاری‌های مختلف و مشاهده گزارش تجمیعی.
- **📊 نمودارهای تحلیلی:** نمایش توزیع سرمایه و روند رشد دارایی‌ها با استفاده از نمودارهای تعاملی.
- **🧾 شفافیت کارمزدها:** رهگیری و محاسبه دقیق کارمزدهای پرداختی برای هر معامله.
- **📱 اپلیکیشن پیشرو (PWA):** قابلیت نصب سیستم به عنوان اپلیکیشن روی گوشی‌های موبایل و دسکتاپ بدون نیاز به اپ‌استور.
- **👥 مدیریت چند کاربره:** امکان تعریف پرتفوی مستقل برای افراد مختلف خانواده در یک سیستم.
- **📄 خروجی گزارشات:** امکان دریافت گزارش‌های جامع به صورت PDF (با فونت فارسی) و Excel.

---

## 🛠 تکنولوژی‌های استفاده شده

- **Back-end:** PHP 8+ (Custom MVC Architecture), PDO
- **Front-end:** HTML5, Tailwind CSS, JavaScript (Vanilla)
- **Database:** MySQL / MariaDB
- **Libraries:** 
  - [Chart.js](https://www.chartjs.org/) (برای رسم نمودارها)
  - [mPDF](https://mpdf.github.io/) (برای تولید خروجی PDF فارسی)
  - [FontAwesome](https://fontawesome.com/) (برای آیکون‌ها)
